# កម្មវិធីកត់ត្រាអតិថិជនហាងកាត់ដេ
កម្មនេះសម្រាប់ជួយម្ចាស់ហាងកាត់ដេសម្លៀកបំពាក់ក្នុងការកត់ត្រាអថិតិជនមកកាត់ដេសម្រៀកបំពាក់។ ដែលកម្មវិធីនេះមានមុខងាដូចខាងក្រោមនេះ៖
### Login
ជួយការពារទិន្ន័យកុំឲ្យអ្នកផ្សេងចូលប្រើប្រាស់បាន៖
![alt](./src/lib/assets/image/login.jpg)
## ទំព័រខាងមុខ
ទំព័រខាងមុខនេះមានងារដូចខាងក្រោមនេះ
### បញ្ចូលទិន្ន័យ
មានប៊ូតុនសម្រាប់ចុចចូល form ដើម្បីបញ្ចូលទិន្ន័យរបស់អតិថិជន នៅក្នុងមុខងារនេះអាចបញ្ចូលលេខរបស់អាវ និងខោ។
![alt](./src/lib/assets/image/form-insert.jpg)
### បង្ហាញទិន្ន័យ
ការបង្ហាញទិន្ន័យនៅទំព័រមុខនេះមានមុខងារជាប្រភេទ list៖
![alt](./src/lib/assets/image/homeview.jpg)
### ស្វែងរក
ក្នុងការស្វែងទិន្ន័យតាមរយឈ្មោះ ឬលេខទូរស័ព្ទ
![alt](./src/lib/assets/image/search.jpg)
